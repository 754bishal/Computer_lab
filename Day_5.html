How to set up for Github?

1. How to create G-mail account?

Steps to create gmail account are:

  1. Go to www.google.com
  2. Click on sign up
  3. Fill your name, desired gmail and a strong password
  4. Verify your phone number and the code sent by google
  5. Either provide additional information or wrap up the process

2. How to create github account?

Steps to create github account are:

  1. Go to https://github.com/join to create a GitHub account.
  2. Enter your personal details.
  3. Click Verify to start the verification puzzle
  4. Click the green Create account button
  5. Verify your email by entering the code

3. How to connnect vs code editor to github act?

Steps to connect vs code editor to github act are:

  1. Download github in your device
  2. Turn on backup in VS code then login to your github account

4. How to clone or pull repository from github?

Steps to clone or pull repository from github are:

  1. After connecting Vs code to gihub account, select the clone repository option from the side menu.
  2. From the dropdown that appears fro the search bar, select the repository that you want to clone
  3. Select address for the repository and click ok

5. How to push repository or folder to github?

Steps to push repository or folder to github are:

  1. Save whatever you did in the cloned repository files.
  2. In the icon showing a slight pop up, click and click on commit.
  3. Click yes in the pop up.
  4. Type your message in the new window of VS code that opens and saves it.
  5. Click on sync in the side bar

6. How to publish the repo as live site?

Steps to publish the repo as live site are:

  1. Upload or create files in public repo
  2. Open your repository in Github and navigate to settings
  3. Click on pages
  4. IN the dropdown menu where it says none, change it to root and click save.
  5. Wait for few minutes and click refresh
  6. At the top of the page, the link to your published site will be shown.
